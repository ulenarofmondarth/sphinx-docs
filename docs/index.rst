Mondarth Sphinx-docs documentation
==================================

This image is used to create documentation accompanying various Mondarth projects.

Pull the image using:

.. code:: bash

   docker pull ghcr.io/ulenarofmondarth/sphinx-docs:latest

Or refer to the `packages <https://github.com/ulenarofmondarth/sphinx-docs/pkgs/container/sphinx-docs>`_ page for explicit version tags.
